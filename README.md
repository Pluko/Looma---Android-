# Looma - Your Data. Your Choice.

A privacy-focused Android dashboard that shows you what permissions your apps have and how you're using them.

**Note:** This is an independent privacy tool. Your data never leaves your device.

## Early Beta

This is a working prototype. Expect rough edges!

## Features

- **Permission Scanner**: See which apps have access to:
  - Camera
  - Microphone
  - Location
  - Contacts
  - Calendar
  - SMS
  - Storage
  - And more

- **Usage Tracking**: 24-hour breakdown of app screen time
- **Privacy Score**: Simple calculation based on permission usage
- **On-Device Only**: No data leaves your phone

## Installation

1. Download the latest APK from [Releases](https://github.com/YOUR-USERNAME/looma-android/releases)
2. Enable "Install from Unknown Sources" in Android Settings
3. Install the APK
4. Grant "Usage Access" permission when prompted

## Requirements

- Android 8.0+
- Usage Access permission (for app usage stats)

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Android PackageManager API
- UsageStatsManager API
